# Intrigue Map

An interactive map for visualising and linking story elements in RPG sessions (characters, places, events, items, organisations, etc.). You can add, edit, and connect nodes and edges to keep your web of relationships and plot threads in one place.
Try it out right now: https://intrigue-map.vercel.app/
